# 📖 PDF Reading Progress

Маленькое окно поверх всех окон. Открываешь рядом с PDF-вьювером и отслеживаешь прогресс.

## Запуск

```
PDF Progress.exe
```

## Использование

1. Запусти `PDF Progress.exe`
2. Введи **«С страницы»** и **«По страницу»** — это твой диапазон задания
3. Автоматический определяется и обновляется страница на которой вы сейчас находитесь
4. Есть возможность ввести текущую страницу вручную (или жми **−** / **+**)
5. Окно всегда поверх других — удобно держать рядом с учебником

## Настройки сохраняются

Диапазон и текущая страница записываются в `settings.json` рядом со скриптом.
При следующем запуске всё восстановится.

## Отключить «поверх всех окон»

Нажми на 📌 в заголовке — окно перестанет быть поверх остальных.
